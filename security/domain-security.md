# Как защитить домен от потери

Для важных доменов включайте базовые меры защиты.

## Рекомендуем

- включить автопродление;
- использовать двухфакторную аутентификацию;
- держать контактный email актуальным;
- не отключать блокировку на перенос без необходимости;
- ограничить доступ сотрудников к аккаунту.

## Для компаний

Создайте отдельный рабочий аккаунт для управления доменами и храните доступы централизованно.
