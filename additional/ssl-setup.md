# Как установить SSL

SSL-сертификат нужен, чтобы сайт открывался по HTTPS.

## Что нужно

- домен должен вести на ваш сайт;
- сервер должен быть доступен из интернета;
- на хостинге должна поддерживаться установка сертификатов.

## Варианты установки

- автоматический SSL на хостинге;
- Let's Encrypt;
- сертификат от стороннего центра сертификации.

## Общий порядок

1. Подключите домен к сайту.
2. Выпустите сертификат через хостинг или сервер.
3. Включите HTTPS.
4. Настройте редирект с HTTP на HTTPS.
5. Проверьте отсутствие смешанного контента.
