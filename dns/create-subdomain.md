# Как создать поддомен

Поддомен — это отдельный адрес внутри основного домена, например `blog.example.com`.

## Как создать

1. Перейдите в DNS-записи домена.
2. Добавьте запись для нужного имени, например `blog`.
3. Выберите тип записи:
   - **A**, если поддомен должен вести на IP;
   - **CNAME**, если поддомен должен указывать на другой домен.
4. Сохраните изменения.

## Примеры использования

- `blog` — блог;
- `mail` — почтовый сервис;
- `app` — веб-приложение;
- `shop` — интернет-магазин.
