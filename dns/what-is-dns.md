# Что такое DNS и для чего он нужен

DNS помогает связать доменное имя с нужным сервером или сервисом.

## Простыми словами

Когда пользователь вводит домен в браузере, DNS подсказывает, куда нужно направить запрос.

## Основные типы записей

- **A** — привязка домена к IPv4-адресу;
- **AAAA** — привязка к IPv6-адресу;
- **CNAME** — псевдоним для другого имени;
- **MX** — маршрутизация почты;
- **TXT** — текстовые записи для подтверждений и настроек;
- **NS** — серверы, которые управляют зоной домена.

## Когда нужен DNS

DNS нужен почти всегда: для сайта, почты, поддоменов, верификации сервисов и CDN.
