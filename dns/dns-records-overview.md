# Как добавить A, AAAA, CNAME, MX, TXT записи

Ниже кратко описано, для чего нужна каждая запись и как её добавить.

## A-запись

Используется для привязки домена к IPv4-адресу сервера.

### Как добавить
1. Откройте DNS-зону.
2. Нажмите **Добавить запись**.
3. Выберите тип **A**.
4. Укажите имя хоста и IP.
5. Сохраните.

## AAAA-запись

Используется для IPv6-адреса.

## CNAME

Используется для создания псевдонима, например `www`.

## MX

Используется для работы почты на домене.

## TXT

Используется для верификации домена, SPF, DKIM и других настроек.

## Советы

- не дублируйте противоречащие записи;
- внимательно проверяйте имя хоста;
- после изменений дождитесь обновления DNS.
