# Как продлить домен

Продление сохраняет право на использование домена на следующий период.

## Как продлить вручную

1. Войдите в личный кабинет.
2. Откройте список ваших доменов.
3. Выберите нужный домен.
4. Нажмите **Продлить**.
5. Выберите срок продления.
6. Оплатите продление.

## Рекомендации

- продлевайте домен заранее;
- включайте автопродление для важных доменов;
- следите за актуальностью способа оплаты.

## Если домен уже истёк

После окончания срока регистрации домен может перейти в льготный период, а затем стать недоступным для обычного продления. Точные условия зависят от доменной зоны.
