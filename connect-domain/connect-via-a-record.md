# Подключение через A-запись

Этот способ подходит, если у вас есть IP-адрес сервера.

## Что понадобится

- IPv4-адрес сервера;
- при необходимости IPv6-адрес;
- доступ к DNS-зонам домена.

## Пошагово

1. Откройте DNS-записи домена.
2. Добавьте запись типа **A** для корневого домена.
3. Укажите IP-адрес сервера.
4. При необходимости добавьте запись **AAAA** для IPv6.
5. Для `www` создайте **CNAME** на корневой домен или отдельную A-запись.
6. Сохраните изменения.

## Проверка

После сохранения дождитесь обновления DNS и проверьте, открывается ли сайт по домену.
